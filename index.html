HTML

  • The head element contains information about the webpage including the meta tag.
  • The body element contains the content of a webpage.
  • Only one h1 tag should be used on a page.

CSS

  • Separation of Concerns is separating different code into different files based on use. Each Section addresses a separte concern.
  • Margin add space outside of the element.
  • Padding adds space inside the element and increases size of element.
  • DRY Code means DO NOT REPEAT YOURSELF.

Git

  • Use git status to determine current branch.
  • git checkout -b branch-name: creates a new branch and switches to branch
  • Use git stash to stash files.
  • Files need to be staged before they can be committed.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instructions that tell a computer how to perform a certain task.
  • A function has two main parts, declare and call.
  • An array is a single variable that holds a group of data.
  • An array is not limited to a single type of data.
  • The location of the data item in the array is called an index and begins at 0.
